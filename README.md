# Fake E-Commerce (React)

A simple **React-based Fake E-Commerce application** that fetches product data from an external API and displays it dynamically on the UI.  
This project was built to understand **React data flow, Context API for global state management, and API integration**, which will be used in my upcoming **Instagram Clone project**.

---

## Project Purpose

Before building my **Instagram Clone**, I created this project to experiment with:

- API data fetching
- Global state management using **Context API**
- Component-based architecture
- Dynamic rendering of product data

This helped me understand how data flows through a React application.

---

## Tech Stack

**Frontend**
- React
- Context API
- JavaScript
- Axios / Fetch
- React Router
- CSS

**API**
- Fake Store API
